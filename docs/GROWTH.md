### Key differentiators / 核心差异点

| vs | This project / 本项目 |
|----|----------------------|
| Subjective "feels smarter" | **12-dimension quantitative score** with evidence |
| One-off testing | **Longitudinal trend tracking** with regression alerts |
| Easy to game | **Anti-gaming probes** with randomized inputs |
| Black-box scoring | **Open rubrics + formulas** fully transparent |
| External API dependency | **Offline by default**, zero external deps |

## Star Conversion Tips / Star 转化技巧

| Strategy | Detail | 说明 |
|----------|--------|------|
| Hero section | Keep README top concise, metric-driven, with badges | README 顶部简洁有力 |
| Real numbers | Always show actual scores, not just descriptions | 展示真实数据 |
| Copy-paste commands | Quick start must be < 4 commands | 快速开始 < 4 条命令 |
| Before/after | Pin a trend comparison showing improvement | 展示前后对比 |
| Release notes | Concrete value per release, not generic text | 具体的发版说明 |
| Social proof | Reference any team or community using it | 引用使用者 |
